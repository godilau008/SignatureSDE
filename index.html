Instructions

  1. Ajouter vos informations tel qu’indiqué sur la page, puis copier la signature en appuyant sur le bouton prévu à cet effet.
  2. Dans Outlook ou Teams, appuyer sur le bouton engrenage ou les 3 points pour vous rendre dans les réglages. Ceux-ci devraient être en haut à droite de la fenêtre
  3. Dans le menu des réglages, appuyer sur le premier onglet compte, puis sur le sous-menu signature et enfin, ajouter un signature
  4. Enfin, collez votre signature copiée à l’aide du raccourci clavier “ctrl+v” dans la zone à cet effet et donner un nom à votre signature à votre discrétion.
  5. Vous pouvez également établir choisir une de vos signatures sauvegardées dans la même fenêtre comme signature par défaut, autrement, vous devrez utiliser l’onglet “insérer” dans outlook pour insérer une de vos signatures sauvegardées.

Avertissement

Par défaut, la signature sur l’application mobile (android et ios) est “Obtenir Outlook pour XX”. Vous devez suivre les mêmes étapes décrites plus bas sur mobile pour changer votre signature et avoir la bonne signature puisque la signature sur le web n’est pas synchronisée avec l’application mobile. Sur mobile, appuyer sur votre icône de profil en haut à gauche de l'écran, puis l’engrenage en bas à gauche pour se rendre dans les paramètres. Une option signature devrait être visible où vous pouvez coller votre signature. À noter, le logo ne s’affiche pas initialement dans l’éditeur de signature, mais devrait fonctionner une fois sauvegardé.
